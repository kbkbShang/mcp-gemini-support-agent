# Software Installation Guide

## Background

Employees install approved software through the internal software portal. Installation failures may happen due to missing permissions, insufficient disk space, network issues, or outdated system packages.

## Installation Steps

1. Open the internal software portal.
2. Search for the required software.
3. Confirm that the software is approved for your role.
4. Click Install.
5. Keep the device connected to the company network or VPN.
6. Restart the computer if required.

## Common Issues

If installation fails with "permission denied", the user may need admin approval.

If installation is stuck, the user should check network connectivity and available disk space.

If the software is not listed, the user may need to request access through the service portal.

## Escalation

Create a support ticket if the software is missing from the portal, installation fails repeatedly, or admin approval is required.
