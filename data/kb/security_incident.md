# Security Incident Reporting

## Background

Employees must report suspicious emails, unauthorized access attempts, or potential security breaches.

## Reporting Steps

1. Do not interact with suspicious content.
2. Capture screenshots if possible.
3. Report the incident immediately.
4. Follow instructions from the security team.

## Common Issues

Phishing emails are the most frequently reported security incidents.

## Escalation

Immediately create a high-priority security ticket.
