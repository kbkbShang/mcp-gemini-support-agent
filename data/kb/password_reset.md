# Password Reset Guide

## Background

Employees may need to reset their password when it expires, is forgotten, or fails during SSO login. A password reset can usually be completed through the self-service identity portal.

## Reset Steps

1. Go to the company identity portal.
2. Click "Forgot Password".
3. Enter your company email address.
4. Complete MFA verification.
5. Create a new password that meets company security requirements.
6. Try signing in again with the new password.

## Password Requirements

The new password must be at least 12 characters long and include uppercase letters, lowercase letters, numbers, and special characters.

The new password cannot be the same as the previous password.

## Escalation

Create a support ticket if the user cannot complete MFA verification, does not receive the reset email, or the identity portal returns an error.
