# Software Update Procedure

## Background

Keeping software up to date helps maintain security and stability.

## Update Steps

1. Open the software portal.
2. Check for available updates.
3. Install approved updates.
4. Restart the application.
5. Reboot the device if required.

## Common Issues

Updates may fail due to insufficient permissions or lack of disk space.

## Escalation

Create a support ticket if updates repeatedly fail or rollback unexpectedly.
