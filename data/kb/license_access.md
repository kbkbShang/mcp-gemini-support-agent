# Software License Access

## Background

Employees may need access to licensed software tools such as development environments, design tools, analytics platforms, or engineering applications. License access is usually controlled by user groups and license servers.

## Access Request Steps

1. Confirm the software name and required license type.
2. Check whether the user belongs to the correct access group.
3. Submit an access request through the internal service portal.
4. Wait for manager approval if required.
5. Restart the software after access is granted.

## Common Issues

If the software shows "license checkout failed", the license server may be unavailable or all licenses may be in use.

If the user receives "permission denied", they may not be part of the required access group.

If the license was recently approved, the user should restart the application or sign out and sign in again.

## Escalation

Create a support ticket if the license server is unreachable, access group membership is missing, or the software still cannot check out a license after approval.
