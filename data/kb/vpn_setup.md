# VPN Setup and Troubleshooting

## Background

Employees use VPN to access internal company resources when working remotely. VPN connection issues are often caused by expired credentials, unstable network connections, outdated VPN clients, or missing multi-factor authentication approval.

## Setup Steps

1. Open the company VPN client.
2. Enter your company email and password.
3. Complete the SSO login process.
4. Approve the multi-factor authentication request.
5. Select the nearest VPN region.
6. Click Connect.

## Common Issues

If the VPN client shows "authentication failed", the user should first verify their SSO credentials and complete MFA approval.

If the VPN connection times out, the user should restart the VPN client and check their internet connection.

If the VPN client is outdated, the user should install the latest version from the internal software portal.

## Escalation

Create a support ticket if the user still cannot connect after resetting credentials, restarting the VPN client, and updating the VPN software.
