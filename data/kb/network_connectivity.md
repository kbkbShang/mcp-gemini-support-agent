# Network Connectivity Troubleshooting

## Background

Network issues may prevent access to internal systems, cloud services, and collaboration tools.

## Troubleshooting Steps

1. Verify internet connectivity.
2. Restart the router if working remotely.
3. Connect to VPN if required.
4. Test access to multiple websites.
5. Restart the computer.

## Common Issues

DNS failures and unstable Wi-Fi connections are common causes of connectivity issues.

## Escalation

Create a support ticket if network access remains unavailable after basic troubleshooting.
