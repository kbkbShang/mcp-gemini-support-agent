# SSO Login Troubleshooting

## Background

Single Sign-On (SSO) allows employees to access multiple internal systems with one company account. SSO failures may happen due to expired passwords, locked accounts, missing MFA approval, or browser session issues.

## Troubleshooting Steps

1. Confirm that the company email address is correct.
2. Reset the password if it has expired.
3. Approve the MFA request on the registered device.
4. Clear browser cache and cookies.
5. Try signing in from an incognito browser window.
6. Contact IT support if the account is locked.

## Common Issues

If the user sees "invalid credentials", they should reset their password and try again.

If the MFA prompt does not appear, the user should check their registered device or re-enroll MFA.

If the account is locked, IT support must unlock it before the user can sign in again.

## Escalation

Create a support ticket if the account is locked, MFA cannot be completed, or login continues to fail after password reset.
