# Multi-Factor Authentication Setup

## Background

Multi-Factor Authentication (MFA) provides additional security for company accounts.

## Setup Steps

1. Sign in to the identity portal.
2. Navigate to Security Settings.
3. Register a mobile device.
4. Install the authenticator application.
5. Scan the QR code.
6. Verify the setup.

## Common Issues

Users may lose access to MFA devices after changing phones.

## Escalation

Create a support ticket if MFA cannot be reconfigured or recovery options are unavailable.
